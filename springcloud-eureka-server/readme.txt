配置文件参数：
	eureka.instance.hostname				
	eureka.client.register-with-eureka		是否将自己注册到 Eureka Server
	eureka.client.fetch-registry			是否从 Eureka Server 获取注册信息
	eureka.client.service-url.defaultZone	设置与 Eureka Server 交互的地址
